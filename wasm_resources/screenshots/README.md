# PWA Screenshots

This directory should contain screenshots for PWA app stores and installation prompts.

## Required Screenshots

1. **gameplay-1.png** (1280x720)
   - Main gameplay screenshot showing 4-player action
   - Highlight weapons and combat

2. **gameplay-2.png** (1280x720)
   - Character customization screen
   - Show NFT hats and skins

3. **gameplay-3.png** (1280x720) [Optional]
   - Map selection screen
   - Show variety of levels

4. **gameplay-4.png** (1280x720) [Optional]
   - Scoring/victory screen
   - Show progression system

## Screenshot Guidelines

- **Resolution**: 1280x720 minimum (16:9 aspect ratio)
- **Format**: PNG for quality
- **Content**: Actual in-game screenshots (not mockups)
- **Annotations**: Optional text overlays highlighting features
- **Branding**: Include "Fishy" branding where appropriate

## How to Capture

1. **In-Game**:
   - Play the game in browser
   - Use browser's screenshot tool or F12 key
   - Capture during exciting moments

2. **Post-Processing**:
   - Crop to 1280x720
   - Add annotations if desired
   - Optimize file size (use tinypng.com)

## Testing

Screenshots appear in:
- PWA install prompts
- App store listings (future mobile apps)
- Social media sharing
- Marketing materials
